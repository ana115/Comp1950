Prototype #1 - Home

  • Visual Appeal
  • Update
    • Since this is that first page the student will encounter it was a priority for the team for it to be llamative, yet professional and functional. So we went for a color scheme that combines warmer colors with colder ones/li>
    • Added visuals to demonstrate key content
    • Fulfilled client request to implement CSS3 and JQuery features.
      • The main center image will slide using the Jquery plugin Nivo Slider
      • A calendar will be added where the instructor can set reminders for the students about important days to avoid confusion
      • The navigation bar on the left will appear and disappear depending on the pointer being locate at the leftmost area of the page
    • Educational content presented in chunks

Prototype #2 - Course Information

  • Visual Appeal
  • Update
    • This page will be using the CSS3 multiple column layout so the student can easily view and access the information about the course during the upcoming weeks
    • Also to make the text books images more interactive the information about them will show up when the mouse is hover over them
    • Fulfilled client request to implement CSS3 and JQuery features.

Prototype #3 - Resources

  • Visual Appeal
  • Update
    • Since the resource page will be less frequently visited than the other pages (given that is not necessary for every class) we took the liberty to make it more image heavy in order to make the content more atractive
    • All the images will link to the respective website of the resource
    • Using Jquery the images will become more opaque when the mouse is hover over them
    • The recommended books images will display more information when the mouse hovers and link to the page to buy them whem cliked

Prototype #4 - Lectures

  • Visual Appeal
  • Update
    • The side bar was completely re-structure. Instead of having a link to all the lectures it will have three different drop down menus, one for the different sections of the lecture, one for exercises and it's solutions and one for homeworks and their solutions after these have been posted.
